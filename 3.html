Имя Достоинства Недостатки
Eternium
  • Уникальная система управления.
  • Одиночная Игра. Хороший аналог синглплеерной классической RPG на мобильном.
  • Есть клиент для PC
  • Озвучка
  • Продуманная схема монетизации (Это мой любимый клиент для просмотра рекламы)
  • Оффлайн прокачка навыков.
  • БОльшая часть контента доступна без доступа к сети.
  • Интересные ивенты. (Прохождение новым персонажем с нуля на скорость на время)
  • Продуманная система крафта.
  • Доверяют данным пользователя, которые могут быть слегка изменены вне клиента.
  • Бесполезное подземелье, для которго присылают по 3 ключа в день. Актуально от силы два часа за всю прокачку.
  • Один город. (0 было хорошо, 4 возможно тоже будет неплохо)
  • Не очень сбалансированы компаньоны.
  • Сейчас ивент. Боюсь, что не допишу это задание...
My Little Pony
  • Много мини-игр
  • Оригинальная озвучка от Hasbro
  • Музыка
  • Пони!
  • Атмосфера
  • Возможность игры оффлайн
  • Хороший потенциал автоматизации
  • Озвучка есть не везде.
  • Можно сломать (Минус скорее для разработчиков, чем для пользователя. тут скорее особенность самой платформы)
PUBG Mobile
  • Сессионная
  • Продуманный геймплей. Решены основные проблемы этого жанра (статичность игры, кемперы)
  • Ф2П без необходимости доната и рекламы
  • Низкий расход траффика (у меня какое-то время был лимит 10г/месяц)
  • Кастомизация персонажа
  • Руки, мозг и опыт (скилл) вместо месяцев прокачки. Можно эффективно играть на новом аккаунте.
  • Уникальные механики (ползание)
  • Красивый интерфейс
